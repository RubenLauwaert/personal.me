# personal.me

Personal portfolio website for Ruben Lauwaert — software engineer and quant researcher based in Mechelen, Belgium.

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and builds
- **Tailwind CSS 4** for styling
- **Radix UI** primitives with shadcn/ui components
- **Lucide** icons

## Features

- Light / dark theme with persistent preference
- Responsive layout across all screen sizes
- Featured projects with live website links
- Experience and education timeline
- Skills overview across technical, financial, and social domains
- Google Calendar integration for booking meetings
- Language proficiency display

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Project Structure

```
src/
  app/
    components/    # Page sections (Hero, Experience, Skills, Projects, etc.)
    components/ui/ # Reusable UI primitives (shadcn/ui)
  styles/          # Global styles, theme variables, fonts
public/            # Static assets (images, favicon)
```
